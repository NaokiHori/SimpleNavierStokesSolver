Case 2 - Conservation of quadratic quantities¶

Check conservation of volume-integrated quadratic quantities \(K\) and \(H\) (see governing equations for the definitions of these quantities).

Configuration¶

Initially we run our solver for \(50\) time units with the following configuration to obtain random velocity and temperature fields.

#!/bin/bash

with_temperature=true \
with_thermal_forcing=true \
timemax=5.0e+1 \
wtimemax=3.0e+2 \
log_rate=1.0e+2 \
log_after=1.0e+2 \
save_rate=1.0e+3 \
save_after=1.0e+3 \
stat_rate=1.0e+0 \
stat_after=1.0e+3 \
ly=1.0e+0 \
itot=32 \
jtot=32 \
stretch=3 \
Ra=1.0e+100 \
Pr=1.0e+0 \
mpirun -n 1 --oversubscribe ./a.out

Notice that \(Ra\) is set to an extremely large value to describe inviscid fluid.

This is followed by another run for \(50\) time units, where the previous simulation is restarted with the thermal forcing off.

#!/bin/bash

with_temperature=true \
with_thermal_forcing=false \
timemax=1.0e+2 \
wtimemax=3.0e+2 \
log_rate=1.0e-1 \
log_after=5.0e+1 \
save_rate=1.0e+3 \
save_after=1.0e+3 \
stat_rate=1.0e+0 \
stat_after=1.0e+3 \
ly=1.0e+0 \
itot=32 \
jtot=32 \
stretch=3 \
Ra=1.0e+100 \
Pr=1.0e+0 \
mpirun -n 1 --oversubscribe ./a.out

Quadratic quantities¶

As being derived in the discrete momentum balance and the discrete thermal energy balance, they should be constant in time in theory. This is not the case in this project since we adopt an explicit Runge-Kutta scheme, which is dissipative (see Morinishi et al., J. Comput. Phys. (143), 1998, Coppola et al., Appl. Mech. Rev. (71), 2019). Thus two quantities should display monotonic decrease in time, which is advantageous from a stability point of view.

See also

Conservations of \(K\) and \(H\) are highly dependent on the discretisation of the advective terms. See a bit detailed analysis.
