Description¶

This function is a wrapper of a function calloc in C standard library stdlib. calloc returns NULL when the allocation fails. This function checks it and output an error message.

Since the failure of memory allocation is fatal, MPI_Abort is called and the simulator is terminated.

 8void *common_calloc(size_t count, size_t size){
 9  void *ptr = calloc(count, size);
10  if(ptr == NULL){
11    fprintf(stderr, "memory allocation error\n");
12    MPI_Abort(MPI_COMM_WORLD, 0);
13  }
14  return ptr;
15}
