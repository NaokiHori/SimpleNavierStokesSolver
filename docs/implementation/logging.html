Description¶

This function computes the divergence of the velocity field and write the results to a file fname.

Local divergence of the velocity field

\[\dder{\ux}{x} + \dder{\uy}{y} = \frac{ \vat{\ux}{\pip, \pjc} - \vat{\ux}{\pim, \pjc} }{\Delta x_{\pic}} + \frac{ \vat{\uy}{\pic, \pjp} - \vat{\uy}{\pic, \pjm} }{\Delta y}\]

defined at cell center \(\left( \pic, \pjc \right)\) is implemented as

51      double ux_xm = UX(i  , j  );
52      double ux_xp = UX(i+1, j  );
53      double uy_ym = UY(i  , j  );
54      double uy_yp = UY(i  , j+1);
55      double div =
56        (ux_xp-ux_xm)/DXF(i)
57       +(uy_yp-uy_ym)/dy;

The maximum value of the local divergence divmax and the summation of them divsum, which should be comparable to machine epsilon, are checked

59      divmax = fmax(divmax, fabs(div));
60      divsum += div;

Finally, after collecting the information among all processes:

64  MPI_Allreduce(MPI_IN_PLACE, &divmax, 1, MPI_DOUBLE, MPI_MAX, MPI_COMM_WORLD);
65  MPI_Allreduce(MPI_IN_PLACE, &divsum, 1, MPI_DOUBLE, MPI_SUM, MPI_COMM_WORLD);

they are written to a file:

67  if(mpirank == 0){
68    FILE *fp = NULL;
69    if(param->step == 0){
70      fp = fileio_fopen(fname, "w");
71    }else{
72      fp = fileio_fopen(fname, "a");
73    }
74    if(fp != NULL){
75      fprintf(fp, "%8.2f % .1e % .1e\n", param->time, divmax, divsum);
76      fileio_fclose(fp);

Note

The output should be done only by the master process (mpirank is 0).

Description¶

This function computes the net momentum in each direction \(\int \ux dx dy\) and \(\int \uy dx dy\) and write them to a file fname.

The discrete forms lead

\[\sum_{i} \sum_{j} \vat{ \left( \ux \Delta x \Delta y \right) }{\xic, \xjc}\]

and

\[\sum_{i} \sum_{j} \vat{ \left( \uy \Delta x \Delta y \right) }{\yic, \yjc}.\]

Note that the summations are taken over all \(\ux\) and \(\uy\) components, respectively.

The implementations are

and

103    for(int i=1; i<=itot; i++){
104      double cellsize = DXF(i)*dy;
105      moms[1] += UY(i, j)*cellsize;
106    }
107  }

respectively.

They are written to a file after being collected among all processes (see check_divergence above).

Description¶

This function computes the instantaneous Nusselt number (heat transfer enhancement by the advective effects) \(Nu \left( t \right)\) based on various definitions and write them to a file fname. We compute \(Nu\) by four different ways.

See also

The continuous and discrete relations are derived in the governing equations and the Nusselt number relations, respectively.

  1. Heat flux (on the walls)

    \[Nu \left( t \right) = -\vat{\ave{\der{T}{x}}{y}}{\text{left wall}} = -\vat{\ave{\der{T}{x}}{y}}{\text{right wall}},\]

    which are discretised as

    \[\begin{split}Nu & = -\frac{1}{l_y} \sum_{j} \vat{\left( \dder{T}{x} \Delta y \right)}{\text{left wall}, \xjc} \\ & = -\frac{1}{l_y} \sum_{j} \vat{\left( \dder{T}{x} \Delta y \right)}{\text{right wall}, \xjc}\end{split}\]

    and implemented as

    192  double nu_wall = 0.;
193  {
194    // line integral on the walls
195    for(int j=1; j<=jsize; j++){
196      double dtdx0 = (TEMP(     1, j)-TEMP(   0, j))/DXC(     1);
197      double dtdx1 = (TEMP(itot+1, j)-TEMP(itot, j))/DXC(itot+1);
198      nu_wall -= 0.5*(dtdx0+dtdx1)*dy;
199    }
200    MPI_Allreduce(MPI_IN_PLACE, &nu_wall, 1, MPI_DOUBLE, MPI_SUM, MPI_COMM_WORLD);
201    // line average
202    nu_wall /= ly;
203  }

  2. Energy injection

    \[Nu \left( t \right) = \sqrt{Pr} \sqrt{Ra} \ave{\ux T}{x,y} + 1\]

    which is discretised as

    \[Nu = \frac{1}{l_x l_y} \sqrt{Pr} \sqrt{Ra} \sum_i \sum_j \vat{ \left( \ux \dintrpa{T}{x} \Delta x \Delta y \right) }{\xic, \xjc} + 1,\]

    which is implemented as

    205  double nu_inje = 0.;
206  {
207    // surface integral
208    for(int j=1; j<=jsize; j++){
209      for(int i=1; i<=itot+1; i++){
210        double integrand = UX(i, j)*0.5*(TEMP(i-1, j)+TEMP(i, j));
211        double cellsize = DXC(i)*dy;
212        nu_inje += sqrt(Pr)*sqrt(Ra)*integrand*cellsize;
213      }
214    }
215    MPI_Allreduce(MPI_IN_PLACE, &nu_inje, 1, MPI_DOUBLE, MPI_SUM, MPI_COMM_WORLD);
216    // surface average
217    nu_inje /= lx*ly;
218    nu_inje += 1.;
219  }

  3. Kinetic energy dissipation

    \[Nu = \sqrt{Pr} \sqrt{Ra} \ave{\epsilon_k}{V,t} + 1\]

    which is discretised as

    \[\begin{split}Nu & = \frac{1}{l_x l_y} \sqrt{Pr} \sqrt{Ra} \sum_{i} \sum_{j} \vat{ \left\{ \frac{\sqrt{Pr}}{\sqrt{Ra}} \left( s_{xx} s_{xx} + s_{xy} s_{xy} \right) \Delta x \Delta y \right\} }{\xic, \xjc} \\ & + \frac{1}{l_x l_y} \sqrt{Pr} \sqrt{Ra} \sum_{i} \sum_{j} \vat{ \left\{ \frac{\sqrt{Pr}}{\sqrt{Ra}} \left( s_{yx} s_{yx} + s_{yy} s_{yy} \right) \Delta x \Delta y \right\} }{\yic, \yjc} \\ & + 1,\end{split}\]

    which is implemented as

    221  double nu_eps_k = 0.;
222  {
223    // surface integral (x-momentum contribution)
224    for(int j=1; j<=jsize; j++){
225      for(int i=1; i<=itot+1; i++){
226        double cellsize = DXC(i)*dy;
227        double integrand = analyses_compute_kinetic_dissipation_x(param, fluid, i, j);
228        nu_eps_k += sqrt(Pr)*sqrt(Ra)*integrand*cellsize;
229      }
230    }
231    // surface integral (y-momentum contribution)
232    for(int j=1; j<=jsize; j++){
233      for(int i=1; i<=itot; i++){
234        double cellsize = DXF(i)*dy;
235        double integrand = analyses_compute_kinetic_dissipation_y(param, fluid, i, j);
236        nu_eps_k += sqrt(Pr)*sqrt(Ra)*integrand*cellsize;
237      }
238    }
239    MPI_Allreduce(MPI_IN_PLACE, &nu_eps_k, 1, MPI_DOUBLE, MPI_SUM, MPI_COMM_WORLD);
240    // surface average
241    nu_eps_k /= lx*ly;
242    nu_eps_k += 1.;
243  }

    The discrete dissipation rate is computed in src/analyses.c.

  1. Thermal energy dissipation

    \[Nu = \sqrt{Pr} \sqrt{Ra} \ave{\epsilon_h}{x,y}\]

    which is discretised as

    \[Nu = \frac{1}{l_x l_y} \sqrt{Pr} \sqrt{Ra} \sum_{i} \sum_{j} \vat{ \left\{ \frac{1}{\sqrt{Pr} \sqrt{Ra}} \left( r_x r_x + r_y r_y \right) \Delta x \Delta y \right\} }{\pic, \pjc},\]

    which is implemented as

    245  double nu_eps_h = 0.;
246  {
247    // surface integral
248    for(int j=1; j<=jsize; j++){
249      for(int i=1; i<=itot; i++){
250        double cellsize = DXF(i)*dy;
251        double integrand = analyses_compute_thermal_dissipation(param, temperature, i, j);
252        nu_eps_h += sqrt(Pr)*sqrt(Ra)*integrand*cellsize;
253      }
254    }
255    MPI_Allreduce(MPI_IN_PLACE, &nu_eps_h, 1, MPI_DOUBLE, MPI_SUM, MPI_COMM_WORLD);
256    // surface average
257    nu_eps_h /= lx*ly;
258  }

    The dissipation rate is computed in src/analyses.c.

Note that these four \(Nu\) should return the same value in a statistical sense.

They are written to a file after being collected among all processes (see check_divergence above).
