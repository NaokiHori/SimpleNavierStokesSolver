temperature/¶

Directory src/temperature contains source files which mainly handle the integration of the temperature field and the computation of the buoyancy force on the momentum equation.

temperature_t¶

A structure temperature_t is defined in include/temperature.h, which contains parameters which are relevant to the scalar field \(T\):

 8struct temperature_t_ {
 9  double *temp;
10  double *tempforcex;
11  double *srctempa, *srctempb, *srctempg;
12};

The meanings of each member are as follows:

  1. temp

    Temperature

  2. tempforcex

    Buoyancy force in \(x\) direction

  3. srctemp[abg]

    Terms in the right-hand-side of the scalar equation (source terms) Suffices a, bg are used to distinguish each term and imply \(\alpha\), \(\beta\), and \(\gamma\), respectively.

Also see here for details, in particular how these terms are used in the code.
