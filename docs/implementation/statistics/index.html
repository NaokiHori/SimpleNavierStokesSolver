statistics/¶

Directory src/statistics contains source files which are used to collect temporally-averaged statistics and to output them.

  • statistics/collect.c
  • statistics/finalise.c
  • statistics/init.c
  • statistics/output.c

statistics_t¶

A structure statistics_t is defined in include/statistics.h, which contains data structures which are used to accumulate temporally-averaged quantities:

 8typedef struct {
 9  int num;
10  double *ux1, *ux2;
11  double *uy1, *uy2;
12  double *temp1, *temp2;
13} statistics_t;

The meanings of each member are as follows:

  1. num

    Number of statistics which have been collected

  2. ux1 and ux2

    Averaged \(x\) velocity field and its squared quantity

  3. uy1 and uy2

    Averaged \(y\) velocity field and its squared quantity

  4. temp1 and temp2

    Averaged temperature field and its squared quantity

See src/statistics/collect.c for details.
