Description¶

\(\ave{q}{t}\), which is a temporally-averaged value of \(q\), is approximated as:

\[\ave{q}{t} = \frac{\int_{t} q dt}{\int_{t} dt} \approx \frac{\sum_{n=0}^{t_N-1} q_n}{N},\]

i.e., \(q\) at different time (\(n\) denotes equidistant simulation time step) should be summed up.

By default, these procedures are considered for fluid->ux, fluid->uy, and temperature->temp:

69  collect_mean_ux(param, parallel, fluid, statistics);
70  collect_mean_uy(param, parallel, fluid, statistics);
71  collect_mean_temp(param, parallel, temperature, statistics);

See below for the detailed explanations of these functions.

Finally the number of samples statistics->num (which corresponds to the number of samples \(N\)) is incremented:

73  statistics->num += 1;
