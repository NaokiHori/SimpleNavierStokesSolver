fluid/correct_velocity.c¶

Correct non-solenoidal velocity \(u_i^*\) to solenoidal (and new-step) one \(u_i^{n+1}\) following \(\frac{u_i^{n+1} - u_i^*}{\gamma \Delta t} = -\frac{\delta \psi}{\delta x_i}\).

The methodology is discussed here.

Description¶

The tentative velocity field \(u_i^*\) does not satisfy the (discrete) divergence-free condition, i.e.,

\[\frac{\delta u_x^*}{\delta x} + \frac{\delta u_y^*}{\delta y} = \frac{\left. u_x^* \right|_{i+\frac{1}{2},j} - \left. u_x^* \right|_{i-\frac{1}{2},j}}{\Delta x_i} + \frac{\left. u_y^* \right|_{i,j+\frac{1}{2}} - \left. u_y^* \right|_{i,j-\frac{1}{2}}}{\Delta y } \ne 0,\]

which is modified by using the scalar potential \(\psi\) obtained by solving a Poisson equation.

In \(x\) direction, we compute

\[\left. u_x^{n+1} \right|_{i+\frac{1}{2},j} = \left. u_x^* \right|_{i+\frac{1}{2},j} - \gamma \Delta t \frac{\left. \psi \right|_{i+1,j} - \left. \psi \right|_{i,j}}{\Delta x_{i+\frac{1}{2}}},\]

which is implemented as

23      double dxi = 1./DXC(i);
24      double psi_xp = PSI(i  , j  );
25      double psi_xm = PSI(i-1, j  );
26      UX(i, j) -= gamma*dt*(psi_xp-psi_xm)*dxi;

In \(y\) direction, we compute

\[\left. u_y^{n+1} \right|_{i,j+\frac{1}{2}} = \left. u_y^* \right|_{i,j+\frac{1}{2}} - \gamma \Delta t \frac{\left. \psi \right|_{i,j+1} - \left. \psi \right|_{i,j}}{\Delta y},\]

which is implemented as

34      double psi_yp = PSI(i  , j  );
35      double psi_ym = PSI(i  , j-1);
36      UY(i, j) -= gamma*dt*(psi_yp-psi_ym)/dy;
