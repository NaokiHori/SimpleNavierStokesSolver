Description¶

This function updates pressure field (from \(p^k\) to \(p^{k+1}\)) following a procedure of the SMAC method.

The pressure correction \(\psi\), which is obtained by solving a Poisson equation, is added to the pressure:

\[p^{k+1} = p^k + \psi,\]

which is implemented as

26  for(int j=1; j<=jsize; j++){
27    for(int i=1; i<=itot; i++){
28      P(i, j) += PSI(i, j);
29    }
30  }

As discussed in the temporal discretisation, when the diffusive terms are treated implicitly, additional term should be taken into account, which is

\[\begin{split}\begin{aligned} & - \frac{\gamma^k \Delta t}{2 Re} \frac{\delta^2 \psi}{\delta x^2}, \\ & - \frac{\gamma^k \Delta t}{2 Re} \frac{\delta^2 \psi}{\delta y^2}, \\ \end{aligned}\end{split}\]

for the implicit treatment in \(x\) and \(y\) directions, respectively. Since Laplacian of \(\psi\) at a cell center \(\left( i, j \right)\) is discretised as

\[\frac{\delta^2 \psi}{\delta x_j \delta x_j} = \frac{\left. \frac{\delta \psi}{\delta x} \right|_{i+\frac{1}{2},j}-\left. \frac{\delta \psi}{\delta x} \right|_{i+\frac{1}{2},j}}{\Delta x_i} + \frac{\left. \frac{\delta \psi}{\delta y} \right|_{i,j+\frac{1}{2}}-\left. \frac{\delta \psi}{\delta y} \right|_{i,j+\frac{1}{2}}}{\Delta y},\]

we have the following implementations:

32  if(implicitx){
33    for(int j=1; j<=jsize; j++){
34      for(int i=1; i<=itot; i++){
35        double dpsidx_xm = (-PSI(i-1, j  )+PSI(i  , j  ))/DXC(i  );
36        double dpsidx_xp = (-PSI(i  , j  )+PSI(i+1, j  ))/DXC(i+1);
37        P(i, j) +=
38          -(gamma*dt)/(2.*Re)*(
39              +(dpsidx_xp-dpsidx_xm)/DXF(i)
40        );
41      }
42    }
43  }

which is needed for the implicit treatments in \(x\) direction, and

45  if(implicity){
46    for(int j=1; j<=jsize; j++){
47      for(int i=1; i<=itot; i++){
48        double dpsidy_ym = (-PSI(i  , j-1)+PSI(i  , j  ))/dy;
49        double dpsidy_yp = (-PSI(i  , j  )+PSI(i  , j+1))/dy;
50        P(i, j) +=
51          -(gamma*dt)/(2.*Re)*(
52              +(dpsidy_yp-dpsidy_ym)/dy
53        );
54      }
55    }
56  }

which is for the implicit treatments in \(y\) direction.

Finally the boundary values of the new pressure field are updated:

58  fluid_update_boundaries_p(param, parallel, p);
