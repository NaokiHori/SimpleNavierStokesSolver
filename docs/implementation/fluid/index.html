fluid_t¶

A structure fluid_t is defined in include/fluid.h, which contains all two-dimensional arrays which are necessary to describe and integrate the flow field:

 8struct fluid_t_ {
 9  double *ux, *uy;
10  double *p, *psi;
11  double *srcuxa, *srcuxb, *srcuxg;
12  double *srcuya, *srcuyb, *srcuyg;
13};

The meanings of each member are as follows:

  1. ux

    Fluid velocity in \(x\) direction (\(\ux\))

  2. uy

    Fluid velocity in \(y\) direction (\(\uy\))

  3. p

    Reduced pressure (\(p\))

  4. psi

    Scalar potential which projects non-solenoidal velocity field to a solenoidal one (\(\psi\))

  5. srcu[xy][abg]

    Terms in the right-hand-side of momentum equations (source terms) in each direction (\(x\) and \(y\)) Suffices a, b, and g are used to distinguish each term and imply \(\alpha\), \(\beta\), and \(\gamma\), respectively.

Also see this page for details, in particular how these terms are used in the code.
