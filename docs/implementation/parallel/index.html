parallel_t¶

A structure parallel_t is defined in include/parallel.h, which contains parameters which are relevant to the parallelisation:

10struct parallel_t_ {
11  int mpisize, mpirank;
12  int ymrank, yprank;
13};

The meanings of each member are as follows:

  1. mpisize

    Number of total processors, which is given in exec.sh as e.g., mpirun -n 64 ./a.out

  2. mpirank

    Process ID to identify each process (i.e., all processes have different numbers)

  3. ymrank and ymrank

    parallel->mpirank of the neighbouring processes

Also see src/parallel/init.c to see how they are initialised.
